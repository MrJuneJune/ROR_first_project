# README

Small Ruby on Rails website to apply for a job in Spiria Toronto.

Requirement

- ruby 2.5.3
- postgres

How to install.

```
~$ git clone
```

Edit database.yml file in the config.


```
~$ gem install bundle

~$ bundle

~$ rails s
```

Description:

Created a demo website where people can sign up and apply for a toronto spiria job.
